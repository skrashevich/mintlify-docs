Mock server emulating Telegram Bot API with LLM-generated responses. Supports Ollama, OpenAI, Claude, and any compatible endpoint.

<Card title="GitHub" icon="github" href="https://github.com/skrashevich/telegram-mock-ai">Source code</Card>

## Use Cases

Development without Telegram, automated testing, load testing with proactive mode, CI integration via Docker.

## Quick Start

```bash
docker compose up -d
```

Bot API: `http://localhost:8081` | Admin API: `http://localhost:8082`
