CLI tool for managing MikroTik RouterOS routing entries based on domain name resolutions. Resolves domains to IPs and creates routes through VPN gateways.

<Card title="GitHub" icon="github" href="https://github.com/skrashevich/goaround-block-mikrotik">Source code</Card>

## Quick Start

```bash
go install github.com/skrashevich/goaround-block-mikrotik@latest
./goaround-block-mikrotik -address 192.168.88.1 -username admin -password pass -gateway l2tp-out1 -domain instagram.com
```
