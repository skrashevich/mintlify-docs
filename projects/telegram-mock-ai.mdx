High-quality WAV to MP3 converter. Pure Go, no CGO. LAME 3.100 transpiled to Go. Supports VBR/CBR, ID3v2 tags, cover art.

<Card title="GitHub" icon="github" href="https://github.com/skrashevich/svk-wav2mp3">Source code</Card>

## Install

```bash
go install github.com/svk/wav2mp3/cmd/wav2mp3@latest
```

## Usage

```bash
wav2mp3 -i song.wav
wav2mp3 -i song.wav --bitrate 320
wav2mp3 -i song.wav --title "Title" --artist "Artist" --cover cover.jpg
```

Supports 8/16/24/32-bit PCM, Mono and Stereo.
