Pure Go encoder and decoder for WebP. No CGO, no libwebp. Supports VP8, VP8L, VP8X with alpha, animation, and ICC/EXIF/XMP metadata. 363 tests.

<Card title="GitHub" icon="github" href="https://github.com/skrashevich/go-webp">Source code</Card>

## Install

```bash
go get github.com/skrashevich/go-webp
```

## Usage

```go
img, _ := webp.Decode(reader)
webp.Encode(writer, img, &webp.Options{Lossy: true, Quality: 80})
```
